# Telegram Bot Генератор Паролей

Простой Telegram-бот для генерации безопасных паролей без базы данных. Работает на Python 3.11 с aiogram 3.x.

## Функционал

- Выбор типа символов: только буквы, буквы + цифры, буквы + цифры + символы
- Выбор длины через кнопки (8, 12, 16, 20 символов)
- Выбор количества паролей (1, 3, 5)
- Проверка силы пароля: оценка слабый/средний/сильный
- Кнопка "Еще раз" для повторной генерации
- Команда /password [длина] для быстрой генерации (буквы + цифры)
- Работает с polling

## Команды

- `/start` - Приветственное сообщение с меню кнопок
- `/password [длина]` - Генерирует пароль заданной длины (по умолчанию 12)
- Кнопки: "Сгенерировать пароль" - выбор длины, "Помощь" - инструкция

## Требования

- Python 3.11+
- aiogram 3.x

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Установите переменную окружения:
   - `BOT_TOKEN` - токен Telegram бота
4. Запустите: `python main.py`

## Структура файлов

- `main.py` - основной код бота (~85 строк)
- `requirements.txt` - зависимости
- `railway.toml` - конфигурация для Railway (опционально)
- `.gitignore` - исключения для Git

Пароли генерируются с помощью модуля secrets для безопасности.
